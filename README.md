# AWT-Calculator
This a Simple Calculator using awt package of JAVA libraries!<br> 
### Steps to follow
<ul>
  <li>Clone the directory</li>
  <li>Open the Folder</li>
  <li>Run Calculator.jar</li>
  <li>Make sure you have proper Java Environment</li>
</ul>
